jQuery虽然正在逐渐被Vue、React等新兴框架所替代，但其拥有非常广大的生态圈，旗下众多功能强大的插件依然能够帮助我们大幅的提高研发效率、开发高质量的WEB应用。

借助FEZ工程框架，我们可以使用ES6/ES7的标准特性开发基于jQuery的项目。

FEZ提倡使用jQuery时代的方式来引入jquery和相关插件。

1、使用Bower安装jQuery或插件
FEZ会自动把Bower安装后的文件引入到html页面，发布代码时可在fez.config.js配置合并规则。
如果您的项目使用了大量的插件建议使用此方式。
2、把jQuery或插件代码放在common目录
FEZ会自动把common目录中的文件引入到html页面，发布代码时可在fez.config.js配置合并规则。
如果您的项目有自行开发或修改过的插件建议使用此方式。
3、把jQuery或插件代码放在verdor目录
把vendor目录中的文件手动引入html页面，发布代码时使用usemin配置合并规则。
4、使用CDN地址引入jquery和插件
可选择公司或公共的cdn地址来引入，浏览器可以长期缓存。
